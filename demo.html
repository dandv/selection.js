Selection.js

Selection.js provides a clean API to access selection.

var sel = selection(document.getElementById('demo'));

Click the textarea: 

sel.cursor();

Click to select some text. If you haven't changed the text, it should select "some".

sel.cursor(5, 9);

Click to get the selected text. If you haven't selected any text, please select something.

sel.text();

Click to replace the selected text.

sel.text('this is a replaced text');

Click to insert some text before the cursor.

sel.prepend('this is a prepend text');

Click to insert some text after the cursor.

sel.append('this is an append text');

Click to get the current line text.

sel.line()
